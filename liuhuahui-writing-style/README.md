# Liuhuahui Writing Style

A Claude Code skill that captures liuhuahui's authentic, conversational, and practical writing voice for creating blog posts, tutorials, and opinion pieces.

## 🎯 Overview

This skill helps Claude write content that matches liuhuahui's distinctive style: direct, conversational, practical, and unafraid to challenge mainstream narratives. It combines critical thinking with genuine care for readers, using colloquial language while maintaining clarity and structure.

## ✨ Key Characteristics

### Voice & Tone
- **Direct and Conversational**: Like chatting with a friend over coffee
- **Critical and Opinionated**: Challenges mainstream narratives and "毒鸡汤"
- **Empathetic and Caring**: Shows genuine concern for reader wellbeing
- **Practical and Action-Oriented**: Focuses on actionable advice and concrete steps

### Writing Style
- Uses "你" and "大家" to address readers directly
- Includes colloquial expressions and internet slang (牛马, 吃香难看)
- Employs rhetorical questions to engage critical thinking
- Ends with encouraging messages
- Provides "练习作业" sections for hands-on practice

### Language Patterns
- **Bold text** for key points
- ==Highlights== for critical keywords
- > Blockquotes for important explanations
- → Arrows for transformations
- Clear hierarchical structure with headings
- Mix of short punchy sentences and longer explanations
